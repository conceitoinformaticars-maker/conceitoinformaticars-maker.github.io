Configurar sincronização na nuvem (Google Apps Script)

Siga os passos abaixo uma única vez. Depois disso, seus dados sincronizam automaticamente entre celular e computador, sem precisar de conta em nenhum serviço externo.

1
Acesse script.google.com com sua conta Google e clique em Novo projeto
2
Apague tudo que tiver no editor e cole o código abaixo:
const SHEET_NAME = 'Assistencia'; function doGet(e) { const sheet = getSheet(); const data = sheet.getRange(1,1).getValue(); return ContentService.createTextOutput(data || '{"ordens":[],"nextId":1}').setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const sheet = getSheet(); sheet.getRange(1,1).setValue(e.postData.contents); return ContentService.createTextOutput('ok'); } function getSheet() { const ss = SpreadsheetApp.openById(PropertiesService.getScriptProperties().getProperty('SHEET_ID')); return ss.getSheetByName(SHEET_NAME) || ss.insertSheet(SHEET_NAME); } function setup() { const ss = SpreadsheetApp.create('Conceito Assistencia Dados'); PropertiesService.getScriptProperties().setProperty('SHEET_ID', ss.getId()); Logger.log('Planilha criada: ' + ss.getUrl()); }
3
No menu superior, clique em Executar e escolha a função setup. Autorize quando pedir.
4
Clique em Implantar > Nova implantação. Tipo: App da Web. Quem pode acessar: Qualquer pessoa. Clique em Implantar e copie a URL gerada.
5
Cole a URL abaixo e clique em Conectar:
